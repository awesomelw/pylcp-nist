# Research Scripts

This folder contains project-specific notebooks, scripts, selected plots, and selected data used for the research work in this fork.

## Contents

- `gaussian_beam/` - standalone Gaussian beam MATLAB files and generated plots.
- `lw_edits_gaussian/` - notebooks exploring Gaussian beam intensity and related laser-field behavior.
- `lw_edits_capture_vel/` - notebooks, Python scripts, selected plots, and selected CSV results for capture-velocity and MOT geometry studies.

The main research paper is stored separately at `../paper/ResearchPaper.pdf` so it is easy to find from the repo root.

## Notes

The CSV and PNG files here are research outputs rather than package source code. Keep new exploratory work grouped by topic, and place final paper-facing figures near the notebooks or scripts that generated them. Bulk sweep folders such as `CSVConsoleSweeps/` and previous-output image dumps are ignored because they can be regenerated.
